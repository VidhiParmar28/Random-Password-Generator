# Random Password Generator 🔐

A random password generator built using HTML, CSS, and JavaScript.

## Features
- Generates random passwords
- Uses uppercase letters
- Uses lowercase letters
- Uses numbers
- Uses symbols

## Technologies Used
- HTML
- CSS
- JavaScript

## Concepts Used
- Arrays
- Functions
- Math.random()
- Math.floor()
- DOM Manipulation
